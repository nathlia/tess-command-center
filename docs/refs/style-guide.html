página exemplo →
Para IA: Sempre use var(--token), não hex direto. A página é majoritariamente branca — use --bg-white como base. O único dark section usa --bg-ink (#111827). Seções quentes usam --bg-warm (#f4f2ef) com bordas --border-tan (#c5b8ab). As duas cores de marca são --bg-teal (#0f4c5c) e --bg-purple (#3f2d6f). Texto primário em light: --text-ink (#111827). Secundário: --text-mid (#6b7280).
